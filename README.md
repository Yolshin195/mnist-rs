# [MNIST-rs](https://yolshin195.github.io/mnist-rs/)

Полнофункциональная реализация нейронной сети на **Rust** для распознавания рукописных цифр из датасета [MNIST](http://yann.lecun.com/exdb/mnist/). Проект демонстрирует применение машинного обучения в Rust с фокусом на производительность и типобезопасность.

## 📋 Описание

**MNIST-rs** — это образовательный и практический проект, который включает:
- **Нейронная сеть с обратным распространением** (backpropagation) для обучения
- **HTTP-сервер** для взаимодействия с моделью
- **WebAssembly интеграция** для работы в браузере

Проект организован как монорепозиторий с несколькими крейтами (crates), каждый отвечает за определённую функциональность.

## 🏗️ Архитектура проекта

### Крейты (Crates)

#### 1. **nn-engine** — Ядро нейронной сети
- Реализация многослойного персептрона
- Алгоритмы обучения и вывода
- Сериализация моделей (Serde + Bincode)
- Поддержка работы как на сервере, так и в WebAssembly

**Основные зависимости:**
- `ndarray` — многомерные массивы и линейная алгебра
- `rand` — генерация случайных чисел для инициализации
- Опциональные: `tokio`, `reqwest`, `zip`, `csv` для серверных операций

#### 2. **http-server** — REST API сервер
- Асинхронный HTTP-сервер на базе **Axum**
- REST endpoints для:
  - Загрузки и сохранения моделей
  - Обучения сети на датасете
  - Инференса (предсказания для новых изображений)
- Web UI с шаблонами **Askama**
- Логирование с **Tracing**

#### 3. **wasm-inference** — WebAssembly интеграция
- Компиляция нейронной сети в WebAssembly
- Работа в браузере без backend зависимостей
- Легкий инференс в фронтенде

## 🚀 Быстрый старт

### Требования
- Rust 1.70+
- Cargo

### Установка и запуск сервера

```bash
# Клонировать репозиторий
git clone https://github.com/Yolshin195/mnist-rs.git
cd mnist-rs

# Запустить HTTP-сервер
cargo run -p http-server

# Или использовать Makefile
make run
```

Сервер будет доступен по адресу `http://localhost:3000`

## 📊 Датасет MNIST

Проект работает с классическим датасетом MNIST:
- **70,000 изображений** рукописных цифр (0-9)
- **28×28 пиксела** в оттенках серого
- **60,000** для обучения, **10,000** для тестирования

Модель автоматически загружает датасет при первом запуске.

## 🎯 Функциональность

### Обучение модели
```bash
# Обучить сеть на полном датасете
make train
```

### Тестирование
```bash
# Запустить тесты
cargo test --all
```

### WebAssembly сборка
```bash
# Компилировать в WebAssembly
cargo build --p wasm-inference --target wasm32-unknown-unknown --release
```
